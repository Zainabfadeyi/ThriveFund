# Submission Asset Checklist

Capture these before submitting the final form:

- `dashboard-overview.png`: organizer dashboard after at least one successful payment.
- `public-campaign-mobile.png`: mobile public page with account number and copy buttons visible.
- `payment-proof.pdf`: downloaded proof for one successful payment.
- `campaign-report.pdf`: exported campaign report with payer names.
- `payout-timeline.png`: completed campaign showing Collected -> Settled -> Payout requested -> Paid out.
- `admin-webhook-health.png`: admin webhook health panel with processed/failed/pending counts.
- `admin-nomba-sync.png`: admin reconciliation or overview page showing Sync with Nomba.

Use these in the README, final submission form, or demo deck to show that the product is live, recoverable, and audit-ready.
