# ThriveFund

Save, collect, and reconcile payments effortlessly — a React web app for Nigerian businesses and communities.

## Project Structure

```
ThriveFund/
└── frontend/    # React + Vite + TypeScript + Tailwind
```

## Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build

```bash
cd frontend
npm run build
npm run preview
```

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Recharts
- Lucide React
